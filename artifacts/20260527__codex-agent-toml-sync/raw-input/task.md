# 原始输入

用户希望在工作区初始化时，把已有 Harness Markdown 自定义代理按 Codex 子代理写法写入工作区 `.codex/agents/xxx.toml`。

参考：

- Codex 子代理文档：`https://www.codex-docs.com/configuration/subagents/`
- 现有代理示例：`/Users/zweizhao/project/current/qingtian-harness/agents/qa/testcase/AGENTS.md`

命名规则：

- `title` 和 `description` 可以使用相同目录名。
- 示例：`agents/qa/testcase/AGENTS.md` 生成 `qa-testcase`。

用户选择快速修复。
